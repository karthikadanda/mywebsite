# XPRINT Solutions - Advanced Printing Business Website

A comprehensive, feature-rich website for T-shirt printing, vinyl printing, and eco-solvent printing services.

## Features

### Core Features
- **Responsive Design**: Fully responsive across all devices
- **Modern UI/UX**: Built with Tailwind CSS and modern design principles
- **SEO Optimized**: Meta tags, semantic HTML, and structured data

### Advanced Features
- **Instant Quote Calculator**: Dynamic pricing based on service type, quantity, colors, and urgency
- **Order Tracking System**: Real-time order status tracking
- **File Upload System**: Drag-and-drop file upload with progress tracking
- **Interactive Gallery**: Lightbox functionality for portfolio items
- **Contact Forms**: Multi-purpose contact and inquiry forms
- **Newsletter Subscription**: Email marketing integration ready
- **Social Media Integration**: Share and follow functionality

### Technical Features
- **Smooth Animations**: CSS animations and JavaScript transitions
- **Mobile Navigation**: Responsive hamburger menu
- **Performance Optimized**: Lazy loading and efficient code structure
- **Accessibility**: ARIA labels and semantic HTML
- **Cross-browser Compatible**: Works on all modern browsers

## Services Covered

1. **T-Shirt Printing**
   - Screen Printing
   - DTG (Direct to Garment)
   - Heat Transfer
   - Embroidery
   - Custom Designs

2. **Vinyl Printing**
   - Stickers & Decals
   - Banner Printing
   - Vehicle Wraps
   - Window Graphics
   - Floor Graphics

3. **Eco-Solvent Printing**
   - Large Format Prints
   - Outdoor Banners
   - Canvas Prints
   - Backlit Displays
   - Eco-Friendly Inks

## File Structure

```
WebsiteusingWindsurf/
|-- index.html          # Main HTML file
|-- script.js           # JavaScript functionality
|-- README.md           # Documentation
|-- style.css           # Custom styles (if needed)
|-- sw.js              # Service Worker (PWA)
```

## Getting Started

1. **Clone or download the files**
2. **Open `index.html` in your browser**
3. **No installation required** - it's a static website

## Customization

### Branding
- Update company name, logo, and contact information
- Modify color scheme (currently purple gradient)
- Add your own images and content

### Pricing
- Update base prices in `script.js` quote calculator
- Modify discount tiers and multipliers
- Add new services as needed

### Contact Information
- Update phone numbers, email addresses
- Modify business hours and location
- Add social media links
